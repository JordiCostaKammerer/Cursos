Tasca per a DAW06.
Detalls de la tasca d'aquesta unitat.
Enunciat.



Configura GIT al teu ordinador, crea un repositori públic a GitHub i connecta'ls.

Des del repositori https://github.com/CarlesCanals/Desplegament-web fes-hi un fork al teu repositori de Github i clona l'aplicació al directori local. Connecta Git a Netbeans.
Instal·la Github Desktop per tal de treballar de diferents maneres i mostra-ne la sincronització entre aquest programari i l’IDE.Treballa de manera simultània amb Netbeans i Github Desktop.
Treballa amb dues dues branques: main i develop. Crea un script, Nou.php, que permeti introduir un nou producte a la base de dades, i un altre, Elimina.php, per tal d’eliminar-ne el producte seleccionat. Empra la base de dades la_meva_botiga. Realitza, almenys, una operació commit, push i pull request per branca comentant el resultat de cada execució.
Mostra l'historial de versions del projecte mitjançant una ordre des de consola i visualitza els canvis realitzats a cada branca.
A partir de Netbeans, documenta els scripts nous. Crea la documentació de l'aplicació amb phpDocumentor. Adjunta la carpeta amb el codi generat que permet mostrar-ne la documentación.
Documentació a GitHub. Genera una petita documentació a Github que expliqui la vostra aplicació final, descrivint les caracterítiques més importants.
Al mateix GitHub podeu trobar algunes plantilles que us serviran de referència:

Plantilla bàsica (https://gist.github.com/PurpleBooth/109311bb0361f32d87a2)
Plantilla avançada (https://github.com/othneildrew/Best-README-Template)
Documentau cada acció amb les captures necessàries i explica cada una de les decisions adoptades.


Criteris de puntuació. Total 10 punts.
La valoració de cada apartat és:

    1.    1 punt
    2.    1 punt
    3.    3 punts
    4.    1 punt
    5.    3 punts
    6.    1 punt

Treball en format PDF

El treball ha d’estar presentat en un document PDF, prèviament realitzat amb un document de text (Microsoft Word o LibreOffice Writer). Ha d’incloure portada, índex i nombre de pàgines automatitzades, títols i subtítols numerats, annexos i bibliografia/webgrafia. 

La portada ha d’incloure, com a mínim, el logo del centre educatiu, el nom del mòdul, el nom de l’activitat, el nom i cognoms de l’alumne i el curs actual. 

L’índex ha de coincidir amb els punts indicats a l’activitat

Tipus de font: Arial a tamany 12; interlineat 1,5; marges 3 cm. Notes, encapçalaments, gràfics i taules: Arial a tamany 10. 

Codi font i línies de comandes: Courier New 11 px

Cites, referències i bibliografia o webgrafia han d’afegir-se amb el sistema APA.

Consells i recomanacions.
Incloeu cada apartat a cadascuna de les vostres respostes. Detallau les respostes de manera estructurada, i intentau raonar i explicar de forma clara les decisions adoptades. Tingueu en compte que per als exercicis pràctics hi ha diverses respostes possibles, essent totes elles vàlides.

Indicacions d'entrega.
Una vegada realitzada la tasca elaboraràs un únic document on figurin les respostes corresponents. L'enviament es realitzarà a través de la plataforma de la manera establerta, i l'arxiu es nomenarà seguint les pautes següents:

llinatge1_llinatge2_nom_SIGxx_Tasca

Assegureu-vos que el nom no contingui la lletra ñ, ç... els titlles ni els caràcters especials estranys. Així, per exemple, l'alumna Begoña Sánchez Mañas para la sexta unidad del MP de DAW, hauria de nomenar aquesta tasca com...

sanchez_manas_begona_DAW06_Tasca
